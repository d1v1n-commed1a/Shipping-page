## ⚙️ 동작 방식(로직 요약)

### 1) 장바구니 상태 저장
- 장바구니는 `Map`으로 관리됩니다.
- Key: 상품명(`data-name`)
- Value: `{ name, price, stock, qty }`

### 2) 선택금액 계산
- 장바구니에 담긴 모든 상품에 대해
  - `price * qty` 를 합산하여 선택금액(baseTotal)을 구합니다.

### 3) 할인 계산 순서
1. 멤버십 할인 적용
2. 멤버십 할인 적용 후 금액 기준으로 구간 할인 적용

### 4) 배송비 적용
- Fast 배송 선택 + 장바구니에 상품이 있는 경우에만 3,000원 추가

### 5) 최종금액 반영
- 위 결과를 종합하여 최종금액을 계산하고,
- 우측 aside의 요약 UI에 즉시 반영됩니다.

- `data-name`: 상품명
- `data-price`: 단가
- `data-stock`: 재고
